Welcome to the Books CRUD FastAPI Application

This application is designed to manage a collection of books using a CRUD (Create, Read, Update, Delete) interface. It is built with FastAPI, a modern, fast (high-performance) web framework for building APIs with Python 3.7+.

Purpose

The purpose of this application is to provide a simple and efficient way to manage book records. It is intended for educational and testing purposes, demonstrating how to build a RESTful API with FastAPI.

Scope

The application allows users to perform the following operations:

  • Create: Add new books to the collection.
  • Read: Retrieve details of books from the collection.
  • Update: Modify existing book records.
  • Delete: Remove books from the collection.

Available APIs

  • GET /books: Retrieve a list of all books.
  • POST /books: Add a new book to the collection.
  • GET /books/{id}: Retrieve details of a specific book by ID.
  • PUT /books/{id}: Update details of a specific book by ID.
  • DELETE /books/{id}: Delete a specific book by ID.
